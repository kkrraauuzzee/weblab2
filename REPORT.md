# Отчёт по лабораторной работе №2

## Тема

Разработка веб-приложения с использованием фреймворка Flask.

## Цель работы

Добавить в Flask-приложение страницы для просмотра данных HTTP-запроса и реализовать форму проверки номера телефона с обработкой ошибок на стороне сервера.

## Выполненные изменения

За основу было взято приложение из лабораторной работы №1. Существующие страницы с постами, шаблоны и стили были сохранены. Для выполнения лабораторной работы №2 были добавлены новые маршруты и шаблоны.

## Отображение данных запроса

В приложение добавлены страницы:

- `/request/args` — отображение параметров URL;
- `/request/headers` — отображение заголовков запроса;
- `/request/cookies` — отображение cookie;
- `/request/form` — форма авторизации и отображение параметров формы.

Для получения данных используются объекты `request.args`, `request.headers`, `request.cookies` и `request.form`.

## Форма авторизации

На странице `/request/form` реализована форма авторизации с полями «Логин» и «Пароль». Форма отправляется методом POST. После отправки введённые данные выводятся на этой же странице в таблице.

## Проверка номера телефона

На странице `/phone` реализована форма для ввода номера телефона. Форма состоит из текстового поля и кнопки «Проверить». Данные отправляются на сервер методом POST.

На сервере выполняются проверки:

1. Номер может содержать только цифры, пробелы, круглые скобки, дефисы, точки и символ `+`.
2. Если номер начинается с `+7` или `8`, он должен содержать 11 цифр.
3. В остальных случаях номер должен содержать 10 цифр.

При неверном количестве цифр выводится сообщение:

```text
Недопустимый ввод. Неверное количество цифр.
```

При наличии недопустимых символов выводится сообщение:

```text
Недопустимый ввод. В номере телефона встречаются недопустимые символы.
```

Для отображения ошибки используются классы Bootstrap `is-invalid` и `invalid-feedback`.

Если номер введён корректно, он преобразуется к формату:

```text
8-***-***-**-**
```

## Примеры корректного ввода

```text
+7 (123) 456-75-90
8(123)4567590
123.456.75.90
```
